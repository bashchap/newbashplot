# Purpose of this file is to provide an array that can be sourced.
# The VROBI array maps the positioning of Braille dots as they are
# defined in their font into a more structured order, creating a 
# simplified and deterministic access to a required pattern.
# 
# E.g.
# In the Braille font unicode definition, each box below represents a Braille dot position:
#
#	-------------	  
# 	|   1 |   8 |	  
#	-------------	  
# 	|   2 |  16 |	  
#	-------------	  
# 	|   4 |  32 |	  
#	-------------	  
# 	|  64 | 128 |	  
# 	------------- 	  
#
# The values in the boxes are bitmask values.
#
# E.g.
# The sum of values 1+2+4+64+128 = 199, which represents an L shape.
# 10240 (decimal of x2800) is the unicode Braille font base start.
# 10240 + 199 = 10439, which is the unicode font character '⣇'.
#
# This mapping is not convenient for determining dot positions when plotting.
# For any pixel mapping, the VROBI array points to the actual system mapping.
# A much more convenient dot to bit pattern would be this:
#
#      0     1
#   -------------	  
# 0 |   1 |  16 |	  
#   -------------	  
# 1 |   2 |  32 |	  
#   -------------	  
# 2 |   4 |  64 |	  
#   -------------	  
# 3 |   8 | 128 |	  
#   ------------- 	  
#
# It's convenient because the Braille patterns exist in a grid of 2 columns and 4 rows.
# With this mapping, it's straight forward to determine what Braille point to select
# with a simple calculation, e.g if a dot was required at column 1 row 2, that maps to 64.
# 10240 + 64 = 10304, which is character '⠠'.
#
# The calculation to do this is straight forward.  If you have a column and row point:
#
# 	bitmap= (2 ** y) << ( x * 4)
#
# If you have an existing bitmask at the same Cursor Addressable Area, you just need to
# bitwise OR it with the new one to generate the new Braille character.
#
# With this in mind, the indexed array VROBI has been created.  
# The index is the bitmask, and the element is the translation/ map/ index to the real unicode
# codepoint in uCodePoint array where the element is the actual unicode character you want.
#
# VROBI = Virtual Raster-Ordered Bit Index	  
# Index = VROBI	  
# Element = Associated Braille character	  
#	  
# VROBI mapping         Character Font Mapping	  
# ------------- 	-------------	  
# |   1 |  16 |  	|   1 |   8 |	  
# ------------- 	-------------	  
# |   2 |  32 |  	|   2 |  16 |	  
# ------------- 	-------------	  
# |   4 |  64 |  	|   4 |  32 |	  
# ------------- 	-------------	  
# |   8 | 128 |  	|  64 | 128 |	  
# -------------  	------------- 	  
#	  
#	        # Referenced char from index
